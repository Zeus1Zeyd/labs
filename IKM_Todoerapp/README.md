Todoer - Менеджер задач

Простое и удобное приложение для управления задачами с поддержкой тегов, приоритетов и многопользовательского режима.

Особенности:
1. Создание, редактирование и удаление задач
2. Система тегов с цветовым кодированием
3. Установка сроков выполнения
4. Три уровня приоритета (низкий, средний, высокий)
5. Многопользовательский режим с регистрацией и входом
6. Фильтрация задач (все, активные, завершенные)
7. PostgreSQL для надежного хранения данных

Приложение запускается с фреймворка Qt
Структура проекта:
  C++ файлы:
  databasemanager.cpp Логика БД
  databasemanager.h 
  taskmodel.cpp Бизнес-логика
  taskmodel.h
  main.cpp Инициализация
  QML файлы:
  Main.qml Главный экран
  AddTaskDialog.qml Диалог добавления
  EditTaskDialog.qml Диалог редактирования
  LoginDialog.qml Диалог входа
  TaskCard.qml Диалог задачи

Приложение использует PostgreSQL в качестве серверной базы данных для хранения пользователей, задач и тегов. При запуске пользователь проходит регистрацию или вход через диалоговое окно QML. После аутентификации TaskModel загружает задачи текущего пользователя из базы через DatabaseManager. Пользователь может создавать новые задачи с указанием названия, описания, срока выполнения, приоритета и тегов. Все операции с задачами (добавление, завершение, редактирование, удаление) выполняются через транзакции SQL для обеспечения целостности данных. QML-интерфейс автоматически обновляется благодаря реактивным свойствам TaskModel. Фильтрация задач по статусу выполняется на стороне C++ для повышения производительности. Приложение поддерживает многопользовательский режим, где каждый пользователь видит только свои задачи. Архитектура четко разделяет логику данных (C++) и представление (QML).
