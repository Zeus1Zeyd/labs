Задача 1

В задании 1 реализовать определение класса (поля, свойства, конструкторы (не менее трех),
перегрузку оператора вывода для вывода полей, заданный метод согласно варианту).
Протестировать все методы, включая конструкторы, все исходные данные вводятся с клавиатуры, не забывайте проверять данные на корректность. 

Название класса - Time
Поля - byte hours byte minutes
Методы - Добавление произвольного количества минут (unsigned int) к объекту типа Time. Результат должен быть типа Time.

Заголовочный файл - time.h
Модульный файл - time.cpp

В задании реализовано 3 конструктора: поля с нулями, конструктор с параметрами, конструктор копирования.
Также 2 метода: перегрузка оператора вывода, добавление минут.

Тесты:
1. Ввод начального времени:
  Часы (0-23): 1
  Минуты (0-59): 2
  Сколько минут добавить? 11
  Начальное время: 01:02
  Добавлено минут: 11
  Итоговое время: 01:13
  Время по умолчанию: 00:00
  Копия начального времени: 01:02
2. Ввод начального времени:
  Часы (0-23): 11l
  Ошибка: введены лишние символы: 'l'
  Часы (0-23):
3. Ввод начального времени:
Часы (0-23): l
Ошибка: 'l' не является целым числом.

Задача 2

В задании 2 добавить к реализованному в первом задании классу указанные в варианте
перегруженные операции. 

Унарные операции:
++ добавление минуты к объекту типа Time
-- вычитание минуты из объекта типа Time
Операции приведения типа:
short int (явная) – результатом является количество часов (минуты
отбрасываются);
bool (неявная) – результатом является true, если часы и минуты не равны
нулю, и falseв противном случае.
Бинарные операции:
+ Time t, беззнаковое целое число (минуты). Результат должен быть типа
Time.
– Time t, беззнаковое целое число (минуты). Результат должен быть типа
Time

Заголовочный файл - Time.h
Модульный файл - Time.cpp

В задании добавлены новые методы реализующие операции. Во всех методах обработан случай с отрицательным результатом.
