Задача 1
Создайте сущность Точка, расположенную на двумерной плоскости, которая описывается:
- Координата Х: число
- Координата Y: число
- Может возвращать текстовое представление вида “{X;Y}”
Необходимо создать три точки с разными координатами и вывести на экран их текстовое
представление.

Заголовочный файл: point.h
Модульный файл: point.cpp

Программа создает 3 объекта "точек" на основе введенных данных пользователя и впоследствии выводит их координаты.

Тесты: Введите координаты трех точек:
Точка 1:
X: 1
Y: 2
Точка 2:
X: 3
Y: 4
Точка 3:
X: 5
Y: 6

Текстовое представление точек:
Точка 1: {1;2}
Точка 2: {3;4}
Точка 3: {5;6}

Задача 2
Создайте сущность Линия, расположенную на двумерной плоскости, которая описывается:
- Координата начала: Точка
- Координата конца: Точка
- Может возвращать текстовое представление вида “Линия от {X1;Y1} до {X2;Y2}”
Для указания координат нужно использовать сущность Точка, разработанную в задании 1.1. Создайте
линии со следующими характеристиками:
1. Линия 1 с началом в т. {1;3} и концом в т.{23;8}.
2. Линия 2, горизонтальная, на высоте 10, от точки 5 до точки 25.
3. Линия 3, которая начинается всегда там же, где начинается линия 1, и заканчивается всегда там
же, где заканчивается линия 2. Таким образом, если положение первой или второй линии
меняется, то меняется и третья линия.
4. После создания всех трех объектов измените координаты первой и второй линий, причем
сделайте это таким образом, чтобы положение третьей линии соответствовало требованиям
пункта 3.
5. Измените координаты первой линии так, чтобы при этом не изменились, координаты третьей
линии.

Заголовочный файл: line.h
Модульный файл: line.cpp

На основе введенных координат создаются объекты "линии", после создания 1 и 2 по условию задачи необходими создать 3 с началом первой линии и концом второй. Это достигается использованием занчений первых двух линий и является новым объектом, поэтому зависимо от изменений первых двух. чтобы выполнить пятое заданием значения по x не изменяем.

Тесты: Введите координаты для Линии 1:
Начальная точка:
X: 1
Y: 3
Конечная точка:
X: 23
Y: 8

Введите координаты для горизонтальной Линии 2:
Введите высоту (Y координата, одинаковая для начала и конца): Y: 10
Введите координату X для начала линии: X начала: 5
Введите координату X для конца линии: X конца: 25

Созданные линии:
Линия 1: Линия от {1;3} до {23;8}
Линия 2: Линия от {5;10} до {25;10}
Линия 3 (связана с 1 и 2): Линия от {1;3} до {25;10}

--- Изменение координат ---
Введите новые координаты для Линии 1:
Новая начальная точка:
X: 1
Y: 3
Новая конечная точка:
X: 1
Y: 8

Введите новые координаты для Линии 2:
Новая высота (Y координата): Y: 3
Новое начало (X координата): X: 4
Новый конец (X координата): X: 7

После изменения:
Линия 1: Линия от {1;3} до {1;8}
Линия 2: Линия от {4;3} до {7;3}
Линия 3 (автоматически изменилась): Линия от {1;3} до {7;3}

--- Создаем новые точки для Линии 1 ---
Введите новые координаты для ЛИНИИ 1 (только для нее):
Конечная точка:
X: 9
Y: 10

После изменения только Линии 1:
Линия 1: Линия от {1;3} до {9;10}
Линия 3 (НЕ изменилась): Линия от {1;3} до {7;3}

Задача 3
Создайте сущность Студент, которая описывается:
- Имя: строка
- Оценки: массив целых чисел.
- Может возвращать текстовое представление вида “Имя: [оценка1, оценка2,…,оценкаN]”
Необходимо выполнить следующие задачи:
1. Создать студента Васю с оценками: 3,4,5.
2. Создать студента Петю и скопировать оценки Васи, присвоив содержимое поля с
оценками Васи полю с оценками Пети.
3. Заменить первую оценку Пети на число 5. Вывести на экран строковое представление
Васи и Пети. Объяснить результат
4. Создать студента Андрея и скопировать ему оценки Васи так, чтобы изменение оценок
Васи не влияло на Андрея.

Заголовочный файл: student.h
Модульный файл: student.cpp

С помощью класса можно создавать объекты студентов и присваивать им оценки. Оценки сохраняются в виде векторов, поэтому если скопировать оценки студента Васи для студента Петя, то при изменении оценок Васи, оценки пети ре поменяются, ведь вектор оценок Пети другой.

тест: 
Создание студента Васи:
Введите оценки для Васи:
Введите количество оценок: 3
Введите оценку 1: 3
Введите оценку 2: 4
Введите оценку 3: 5

Создан студент Вася: Вася: [3, 4, 5]

2. Создание студента Пети (копирование оценок Васи):
Создан студент Петя: Петя: [3, 4, 5]

3. Замена первой оценки Пети:
Введите новую первую оценку для Пети: Новая оценка: 5

После изменения первой оценки Пети:
Вася: Вася: [3, 4, 5]
Петя: Петя: [5, 4, 5]

4. Создание студента Андрея (глубокое копирование):

Введите НОВЫЕ оценки для Васи: 
Введите количество оценок: 2
Введите оценку 1: 1
Введите оценку 2: 2

После изменения оценок Васи и создания Андрея:
Вася: Вася: [1, 2]
Андрей: Андрей: [1, 2]
